概率论与数理统计

大数定律与中心极限定理

  • 切比雪夫不等式：$P{|X-EX| \geq \varepsilon } \leq \frac {DX} {\varepsilon ^2}$

  • 大数定律

    • 贝努利大数定理：$\lim _{n \rightarrow \infty} P{|\frac {Y_n} n - p| \geq \varepsilon } = 0$（$Y_n$是n重贝努利试验中成功的次数）
    • 切比雪夫大数定理：$\lim {n \rightarrow \infty} P{|\frac 1 n\sum{i=1}^n X_i - \frac 1 n \sum_{i=1}^n EX_i| \geq \varepsilon } = 0$（$X_1,X_2,\cdots,X_n$是相互独立的随机变量序列）
    • 辛钦大数定理：$\lim {n \rightarrow \infty} P{|\frac 1 n\sum{i=1}^n X_i - \mu| \geq \varepsilon } = 0$（$X_1,X_2,\cdots,X_n$是独立同分布的随机变量序列）

  • 中心极限定理

    • 独立同分布的中心极限定理：$\lim_{n \rightarrow \infty} P{ \frac 1 {\sqrt{n}\sigma}(\sum_{i=1}^n X_i - n\mu) \leq x } = \int_{-\infty}^x \frac 1 {\sqrt{2 \pi}}e^{-t^2/2}\mathrm{d}t$（$X_1,X_2,\cdots,X_n$是独立同分布的随机变量序列）
    • 德·莫佛-拉普拉斯定理：$\lim_{n \rightarrow \infty} P{\frac {Y_n-np} {\sqrt{npq}} \leq x } = \int_{-\infty}^x \frac 1 {\sqrt{2 \pi}}e^{-t^2/2}\mathrm{d}t$（$Y_n$是n重贝努利试验中成功的次数）
    • 林德贝格定理

  • 特征函数

    • 特征函数：$g_X(t) = E(e^{itX})$为随机变量X的特征函数
    • 傅里叶变换：$F(t)=\frac 1 {2\pi} \int_{-\infty}^{+\infty} e^{-itx}f(x) \mathrm{d}x$
    • 性质：
      • $Y=aX$，则$g_Y(t)=g_X(at)$
      • $g_{X+Y}=g_X \cdot g_Y$

数理统计的基本概念

  • 简单随机抽样
  • 经验分布函数
  • 三大抽样分布
    • $\chi^2$（卡方）分布
      • 定义：$X_i \sim N(0,1)$独立同分布，$\chi ^2 = \sum_{i=1}^n X_i^2$服从自由度为n的$\chi^2$分布，记作$\chi^2 \sim \chi^2(n)$
      • 性质：
        • 若$\chi^2 \sim \chi^2(n)$，则$E\chi^2 = n$，$D\chi^2 = 2n$
        • 若$X \sim \chi^2(n_1),Y \sim \chi^2(n_2)$且X, Y相互独立，则$X+Y \sim \chi^2(n_1+n_2)$
    • t分布
      • 定义：$X \sim N(0,1),Y \sim \chi^2(n)$相互独立，$T=\frac {X} {\sqrt{Y/n}}$服从自由度为n的t分布，记作$T \sim t(n)$
    • F分布
      • 定义：$X \sim \chi^2(n_1),Y\sim\chi^2(n_2)$相互独立，$F=\frac {X/n_1} {Y/n_2}=\frac{n_2X}{n_1Y}$服从第一自由度为$n_1$，第二自由度为$n_2$的F分布，记作$F\sim F(n_1,n_2)$
    • $\alpha$分位数：使得$P{X \geq k }=\alpha$的k
  • 统计量
    • 样本均值$\overline{X} = \frac 1 n \sum_{i=1}^n X_i$
    • 样本方差$S^2 = \frac 1 {n-1} \sum_{i=1}^n (X_i - \overline{X})^2$
    • 样本标准差（ 样本均方差）$S = \sqrt{S^2}$
    • 样本k阶原点矩$a_k = \frac 1 n \sum_{i=1}^n X_i^k,\ k=1,2,\cdots$
    • 样本k阶中心矩$A_k = \frac 1 n \sum_{i=1}^n (X_i-\overline{X})^k,\ k=1,2,\cdots$
    • 顺序统计量
    • 样本中位数
    • 样本极差
  • 抽样分布
    • （设$X_1,X_2,\cdots,X_n$为一组来自总体$N(\mu,\sigma^2)$的样本）
      • 样本均值为$\overline{X}$，满足$\overline{X} \sim N(\mu, \frac {\sigma^2} n)$或$\frac {\overline{X}-\mu} {\sigma/\sqrt{n}} \sim N(0,1)$
      • 样本方差为$S^2$，满足$\frac{n-1}{\sigma^2} S^2 = \frac {1} {\sigma^2}\sum_{i=1}^n(X_i - \overline{X})^2 \sim \chi^2(n-1)$
      • 样本均值与样本方差相互独立
      • $\frac {\overline{X}-\mu} {S/\sqrt{n}} \sim t(n-1)$
    • （设$X_1,X_2,\cdots,X_n$和$Y_1,Y_2,\cdots,Y_n$分别是来自总体$N(\mu_1,\sigma^2)$和$N(\mu_2,\sigma^2)$的两组样本，且相互独立）
      • 样本均值差满足$\frac {\overline{X} - \overline{Y}-(\mu_1-\mu_2)} {S_w\sqrt{1/n_1 + 1/n_2}} \sim t(n_1+n_2-2)$，其中$S_w = \sqrt{\frac {(n_1-1)S_1^2 + (n_2-1)S_2^2} {n_1+n_2-2}}$
      • 样本方差比满足$\frac {S_1^2/\sigma_1^2} {S_2^2/\sigma_2^2}\sim F(n_1-1,n_2-1)$
    • 设$X_1,X_2,\cdots,X_n$为一组来自总体$X$的样本，$EX=\mu$，$DX=\sigma^2 < +\infty$.
      • 当样本容量充分大时，近似有$\frac {\sum_{i=1}^n X_i - n\mu} {\sqrt{n}\sigma} \sim N(0,1)$

参数估计

  • 点估计
    • 矩估计法：把未知参数$\theta$表示为总体（原点）矩的函数，再用样本矩估计总体矩，进而就可以用样本矩近似替代总体矩得到$\hat{\theta}$作为未知参数$\theta$的估计。
    • 极大似然估计法：设似然函数$L = \prod_{i=1}^n p(x_i;\theta)$，当给定一组样本$x_1,x_2,\cdots,x_n$时，求出使得似然函数取最大值的$\theta=\hat{\theta}$，$\hat{\theta}$就作为未知参数$\theta$的估计。实际计算时通过微积分方法，解方程$\frac {\mathrm{d}} {\mathrm{d}\theta}\ln L = 0$得驻点后确定最大值。
    • 估计量优良性的标准
      • 无偏性：若$E \hat{\theta} = \theta$，则称$\hat{\theta}$为$\theta$的无偏估计量。（若相互独立地重复多次用无偏估计量进行实际估计，所得估计值的算术平均值与未知参数的真值基本上相同）
      • 有效性：若$\hat{\theta}_1, \hat{\theta}_2$都是$\theta$的无偏估计，若$D\hat{\theta}_1 \leq D\hat{\theta}_2$，则称$\hat{\theta}_1$比$\hat{\theta}_2$有效。若对固定样本容量，$D\hat\theta$达到最小值，则称$\hat\theta$为$\theta$的有效估计量。（$\hat\theta_1$比$\hat\theta_2$在$\theta$附近取值的概率大）
      • 一致性：如果$\hat\theta_n =\hat\theta_n(X_1,X_2,\cdots,X_n)$依概率收敛于$\theta$，则称$\hat\theta_n$是$\theta$的一致估计量。（样本容量越大，越能精确估计出未知参数）
  • 区间估计
    • 对未知参数$\theta$，找到两个统计量$\hat\theta_1,\hat\theta_2$使得对给定$\alpha$有$P{\hat\theta_1 < \theta < \hat\theta_2} = 1-\alpha$，则称$(\hat\theta_1 < \theta < \hat \theta_2)$为$\theta$的置信区间，$1-\alpha$为置信度，$\hat\theta_1$和$\hat\theta_2$分别为置信下限和置信上限。
    • $P{\hat\theta_1 < \theta < \hat\theta_2} = 1-\alpha$表示的含义是随机区间$(\hat\theta_1,\hat\theta_2)$包含$\theta$的概率，而不能理解成未知参数落在区间$(\hat\theta_1,\hat\theta_2)$的概率。
    • 单个正态总体参数的区间估计
      • $\sigma^2$已知，$\mu$的置信区间$(\overline x - u_{\alpha/2} \frac \sigma {\sqrt{n}},\overline x + u_{\alpha/2} \frac \sigma {\sqrt{n}})$
      • $\sigma^2$未知，$\mu$的置信区间$(\overline x - t_{\alpha/2}(n-1) \frac S {\sqrt{n}},\overline x + t_{\alpha/2}(n-1) \frac S {\sqrt{n}})$
      • $\sigma^2$的置信区间$(\frac {(n-1)S^2} {\chi^2_{\alpha/2}(n-1)}, \frac {(n-1)S^2} {\chi^2_{1-\alpha/2}(n-1)})$
    • 两个正态总体参数的区间估计
      • 已知$\sigma_1^2=\sigma_2^2$，$\mu_1-\mu_2$的置信区间$(\overline x - \overline y - t_{\alpha/2}(n_1+n_2-2)S_w\sqrt{\frac 1 {n_1} + \frac 1 {n_2}},\overline x - \overline y + t_{\alpha/2}(n_1+n_2-2)S_w\sqrt{\frac 1 {n_1} + \frac 1 {n_2}})$，其中$S_w = \sqrt{\frac {(n_1-1)S_1^2 + (n_2-1)S_2^2} {n_1+n_2-2}}$
      • $\sigma_1^2/\sigma_2^2$的置信区间$(\frac {S_1^2/S_2^2} {F_{\alpha/2}(n_1-1,n_2-1)},\frac {S_1^2/S_2^2} {F_{1-\alpha/2}(n_1-1,n_2-1)})$
