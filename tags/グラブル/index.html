【日中翻訳練習】Sky Journey - 東山奈央

眺望着那没有星辰的遥远彼方

天空在讴歌着崭新旅途的开端

在陌生的广阔天空之中拥抱着孤独

开始轮回的命运之门悄然打开

如今身处于这遥远之地　身处于这片声音也无法传递的天空

即使如此仍在伸出双手

与你羁绊相连

湛蓝天空

寻求着希望

相遇与离别循环往复

与命运擦肩而过

总有一天能够彼此相连

遥远天空

一直

相信着未来

将无法斩断的记忆

与过去的罪名

就这样怀在内心

继续着前行的步伐

直到无尽旅途的尽头

胸怀着恐惧

即使背对战火的那天来临

我们的羁绊仍存在于此

总有一天能够与你相连

湛蓝天空

越过风儿

挥动羽翼空中翱翔
