工数笔记（二）——极限与连续：数列的极限

数列的概念

  • 一般地，我们把定义在正整数集上的函数\(x_n=f(n)\)称为数列或整标函数。
  • 数列中的每个数叫做数列的项，具有代表性的第\(n\)项\(x_n\)叫做数列的通项或一般项。
  • 几何上，把数列\(\{ x_n \}\)视为跳动的点在数轴\(x\)上的足迹。

数列的极限

直观描述

  • 若随着\(n\)的无限变大，\(x_n\)无限地接近某一常数\(a\)，则说数列\(\{x_n\}\)收敛，并称\(a\)为数列的极限。

  • 若随着\(n\)的无限变大，\(x_n\)不趋近于任何确定的常数，则说数列\(\{x_n\}\)发散。

ε-N定义

设\(a\)为常数，若对任意给定的正数\(\epsilon\)，都存在相应的正整数\(N\)，使得当\(n>N\)时，恒有\(|x_n-a|<\epsilon\)，则说数列\(\{x_n\}\)有极限（或收敛），极限值为\(a\)，记为 \(\lim_{n \rightarrow \infin}x_n=a\)，或简记为\(x_n \rightarrow a \space(n \rightarrow \infin)\).

即\(\lim_{n \rightarrow \infin}x_n=a \Leftrightarrow \forall \epsilon > 0, \exist N > 0\)，使得\(n>N\)时，恒有\(|x_n-a|<\epsilon\).

注意： 1. 数列极限时数列\({x_n}\)变化的最终趋势，所以，任意改变数列中的有限项不影响它的极限值； 2. 定义中的任意性（小）性时十分重要的，否则\(|x_n-a|<\epsilon\)就表达不出无限接近\(a\)的含义； 3. \(N\)与给定的\(ε\)有关，一般地说，\(\epsilon\)越小，\(N\)将越大，它标示变化的进程。

如何证明极限

通过解不等式\(|x_n-a|<ε\)，推出\(n\)应该大于怎样的整数\(N\). 为简化运算，通常可以适当放缩。

例：证明\(\lim _{n \rightarrow \infin} \frac {n+1} n = 1\)

分析：\(\forall \epsilon > 0, \exist N > 0, s.t.当n>N时\) \[ \begin{align} |\frac {n+1} {n} - 1| &= \frac 1 n < \epsilon \\ n &> \frac 1 \epsilon \Rightarrow N=[\frac 1 \epsilon]+1 \end{align} \] 证明：\(\forall \epsilon > 0\)，取\(N=[\frac 1 \epsilon]+1\)，当\(n>N\)时有 \[ |\frac {n+1} n - 1|=\frac 1 n < \epsilon \] 所以数列\(\{ \frac {n+1} n \}\)极限为1.

Q.E.D

定理

  • 在数列\(\{ x_n \}\)中依次任意抽出无穷多项\(x_{n_1},x_{n_2},\cdots,x_{n_k},\cdots\).（其下标\(n_1<n_2<\cdots<n_k<\cdots\)）所构成的新数列\(\{ x_{n_k} \}\)叫做数列\(\{x_n\}\)的子数列。其中\(x_{n_k}\)是原数列的第\(n_k\)项，是子数列的第\(k\)项，显然\(k \leq n_k\).
  • 数列\(\{x_n\}\)收敛于\(a\)的充要条件是它的所有子数列\(\{ x_{n_k} \}\)均收敛于\(a\).
  • 数列\(\{x_n\}\)的奇子数列\(\{x_{2k-1}\}\)和偶子数列\(\{x_{2k}\}\)均收敛于同一常数\(a\)时，则\(\{x_n\}\)也收敛于\(a\).
