无穷级数

  • 概念

  • 敛散性

    • 级数的部分和数列极限存在，则级数收敛，否则级数发散。
    • 等比级数$\sum_{n=1}^{\infty}ar^{n-1}\quad(a\neq 0)$当$|r|<1$时收敛，当$|r| \geq 1$时发散
    • 调和级数$\sum_{n=1}^{\infty}1/n$发散
    • 性质：
      • 级数每项乘以同一非零常数得到的新级数敛散性不变。在收敛的情况，新级数和等于原级数和乘以该常数。
      • 两个收敛级数逐项相加（减）得到的新级数也收敛，且其和与原级数和相等。
      • 在一个级数中，任意去掉、增加或改变有限项后，级数的敛散性不变。但对于收敛级数，其和将受到影响。
      • 在收敛级数内任意加括号得到的新级数也收敛，且其和与原级数和相等。在发散级数内任意去括号得到的新级数也发散。
      • 收敛级数的一般项必趋于零。（一般项不趋于零的级数必发散。）

  • 正项级数

    • 正项级数收敛的充要条件是部分和数列有上界。
    • 比较判别法：设$\sum_{n=1}^\infty u_n$和$\sum_{n=1}^\infty v_n$为两个正项级数，若存在常数$C>0$和正整数$N$，使得$n\geq N$时，恒有$u_n \leq Cv_n$. 则当级数$\sum_{n=1}^\infty v_n$收敛时，级数$\sum_{n=1}^\infty u_n$也收敛；当级数$\sum_{n=1}^\infty u_n$发散时，级数$\sum_{n=1}^\infty v_n$也发散。
    • 比较判别法的极限形式：设$\sum_{n=1}^\infty u_n$和$\sum_{n=1}^\infty v_n$为两个正项级数，若$\lim_{n \rightarrow\infty} \frac {u_n} {v_n}=C$. 则（1）当$0<C<+\infty$时，两个级数敛散性一致；（2）当$C=0$时，若级数$\sum_{n=1}^\infty v_n$收敛时，级数$\sum_{n=1}^\infty u_n$也收敛；（3）当$C=+\infty$时，若级数$\sum_{n=1}^\infty v_n$发散时，级数$\sum_{n=1}^\infty u_n$也发散。
    • P-级数$\sum_{n=1}^\infty 1/n^P$当$P \leq 1$时发散，当$P>1$时收敛。
    • 比值判别法：对正项级数$\sum_{n=1}^\infty u_n$，若$\lim_{n\rightarrow \infty} \frac {u_{n+1}} {u_n} = \rho$，则当$\rho<1$时级数收敛，当$\rho>1$或$\rho=+\infty$时级数发散，当$\rho=1$时无法确定级数敛散性。
    • 根值判别法：对正项级数$\sum_{n=1}^\infty u_n$，若$\lim_{n\rightarrow \infty} \sqrt[n] {u_n} = \rho$，则当$\rho<1$时级数收敛，当$\rho>1$或$\rho=+\infty$时级数发散，当$\rho=1$时无法确定级数敛散性。
    • 积分判别法：对正项级数$\sum_{n=1}^\infty u_n$，函数$f(x)$在区间$[a,+\infty)\quad (a>0)$上非负、连续、单调下降，且$f(n)=u_n\quad(n \geq N)$，则级数$\sum_{n=1}^\infty u_n$与反常积分$\int_{a}^{+\infty} f(x)\mathrm{d}x$敛散性相同。

  • 任意项级数

    • 概念：绝对收敛、条件收敛
    • 莱布尼茨判别法：若交错级数$\sum_{i=1}^\infty (-1)^n u_n$满足条件（1）$\lim_{n \rightarrow \infty} u_n=0$；（2）$u_n \geq u_{n+1},n=1,2,\cdots$。则级数收敛，且其和$S \leq u_1$，余和$r_n=S-S_n$的绝对值$|r_n| \leq u_{n+1}$。
    • 讨论任意项级数的敛散性：
      • 先考察是否绝对收敛；
      • 若不是绝对收敛，再看是否条件收敛；
      • 若用比值法或根值法判定级数不是绝对收敛时，级数的通项不趋于0，故级数必定发散；
      • 对交错级数，可以使用莱布尼茨判别法，还可以利用无穷级数收敛的定义或性质讨论敛散性。
    • 绝对收敛级数的性质：
      • 若级数绝对收敛，且其和为S，则任意交换其各项的次序后所得到的新级数（称为原级数的更序级数）也绝对收敛，其和亦为S。
      • 若级数$\sum_{n=1}^\infty u_n$, $\sum_{n=1}^\infty v_n$都绝对收敛，它们的和分别为S和σ，则它们的柯西乘积$(\sum_{n=1}^\infty u_n)(\sum_{n=1}^\infty v_n)=\sum_{n=1}^\infty (u_1v_{n}+u_2v_{n-1}+\cdots+u_nv_1)$也是绝对收敛的，其和为$S\sigma$。

  • 函数项级数

    • 概念：收敛（发散）点、收敛（发散）域

    • 一致收敛

      • 定义
      • 维尔斯特拉斯M-检定法：如果有收敛的正项常数项级数$\sum_{n=1}^\infty M_n$，使得当$x\in I$时，$|u_n(x)| \leq <M_n\quad(n=1,2,\cdots)$，则函数项级数$\sum_{n=1}^\infty u_n(x)$在区间I上一致收敛。级数$\sum_{n=1}^\infty M_n$称为控制级数或优级数。
      • 性质：
        • 函数项级数的和函数的连续性：在区间$[a,b]$上一致收敛的级数，若级数的每一项都在区间$[a,b]$上连续，则和函数$S(x)$也在区间$[a,b]$上连续。
        • 在区间$[a,b]$上一致收敛的级数，若每一项在$x_0\quad(x_0 \in [a,b])$处都有极限，则和函数在$x_0$处的极限值等于每一项取极限后的级数和。
        • 函数项级数的逐项积分性：在区间$[a,b]$上一致收敛的级数，若级数的每一项都在区间$[a,b]$上连续，则和函数可积，且可逐项积分。逐项积分后的级数也在区间$[a,b]$上一致收敛。
        • 函数项级数的逐项微分性：在区间$[a,b]$上收敛的级数，若级数的每一项都在区间$[a,b]$上有连续的导数，且逐项求导后的级数在区间$[a,b]$上一致收敛，则原级数在区间$[a,b]$上一致收敛，并且可逐项求导。

    • 幂级数

      • 概念：收敛半径，收敛区间
      • 阿贝尔引理：如果幂级数在点$x_0$处收敛，则对$(-|x_0|,|x_0|)$内任意一点都绝对收敛；如果幂级数在点$x_0$处发散，则对$(-\infty,-|x_0|)\cup(|x_0|,+\infty)$内任意一点都发散。
      • 求收敛域：
        • 对幂级数$\sum_{n=0}^\infty a_nx^n$，收敛半径$R=\lim_{n \rightarrow \infty} |{a_n}/{a_{n+1}}|$或$R=\lim_{n \rightarrow \infty} 1/\sqrt[n]{|a_n|}$。为了确定收敛域，还要讨论$x=\pm R$时的两个数项级数是否收敛。
        • 对于缺项的幂级数，可以作变换化为标准幂级数，求得新幂级数的收敛域后再通过解不等式得到原幂级数的收敛域。还可以直接通过定义，用数项级数的判别法确定收敛域。
      • 幂级数的运算：
        • 设有两个幂级数$\sum_{n=0}^\infty a_nx^n$和$\sum_{n=0}^\infty b_nx^n$，收敛半径分别为A, B。记$R=\min { A,B}$。当$A \neq B$时，运算所得级数的收敛半径就是R；当$A=B$时，收敛半径不小于R。
        • 加减法：$(\sum_{n=0}^\infty a_nx^n) \pm (\sum_{n=0}^\infty b_nx^n) = \sum_{n=0}^\infty (a_n \pm b_n)x^n$
        • 乘法：$(\sum_{n=0}^\infty a_nx^n) (\sum_{n=0}^\infty b_nx^n) = \sum_{n=0}^\infty (\sum_{i=0}^n a_ib_{n-i})x^n$
      • 幂函数的分析性质：
        • 幂级数在收敛区间内任一闭子区间上时一致收敛的。
        • 在收敛域上，幂级数的和函数连续。
        • 在收敛域上，幂级数可逐项积分，且收敛半径不变。
        • 在收敛域上，幂级数可逐项微分，且收敛半径不变。

  • 函数的幂级数展开

    • 泰勒级数展开
      • 泰勒级数：$\sum_{n=0}^\infty \frac {f^{(n)}(x_0)} {n!} (x-x_0)^n$
      • 麦克劳林级数：$\sum_{n=0}^\infty \frac {f^{(n)}(0)} {n!} x^n$
