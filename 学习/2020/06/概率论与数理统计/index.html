大数定律与中心极限定理

  • 切比雪夫不等式：$P{|X-EX| \geq \varepsilon } \leq \frac {DX} {\varepsilon ^2}$

  • 大数定律

    • 贝努利大数定理：$\lim _{n \rightarrow \infty} P{|\frac {Y_n} n - p| \geq \varepsilon } = 0$（$Y_n$是n重贝努利试验中成功的次数）
    • 切比雪夫大数定理：$\lim {n \rightarrow \infty} P{|\frac 1 n\sum{i=1}^n Xi - \frac 1 n \sum{i=1}^n EX_i| \geq \varepsilon } = 0$（$X_1,X_2,\cdots,X_n$是相互独立的随机变量序列）
    • 辛钦大数定理：$\lim {n \rightarrow \infty} P{|\frac 1 n\sum{i=1}^n X_i - \mu| \geq \varepsilon } = 0$（$X_1,X_2,\cdots,X_n$是独立同分布的随机变量序列）

  • 中心极限定理

    • 独立同分布的中心极限定理：$\lim{n \rightarrow \infty} P{ \frac 1 {\sqrt{n}\sigma}(\sum{i=1}^n Xi - n\mu) \leq x } = \int{-\infty}^x \frac 1 {\sqrt{2 \pi}}e^{-t^2/2}\mathrm{d}t$（$X_1,X_2,\cdots,X_n$是独立同分布的随机变量序列）
    • 德·莫佛-拉普拉斯定理：$\lim{n \rightarrow \infty} P{\frac {Y_n-np} {\sqrt{npq}} \leq x } = \int{-\infty}^x \frac 1 {\sqrt{2 \pi}}e^{-t^2/2}\mathrm{d}t$（$Y_n$是n重贝努利试验中成功的次数）
    • 林德贝格定理

  • 特征函数

    • 特征函数：$g_X(t) = E(e^{itX})$为随机变量X的特征函数
    • 傅里叶变换：$F(t)=\frac 1 {2\pi} \int_{-\infty}^{+\infty} e^{-itx}f(x) \mathrm{d}x$
    • 性质：
      • $Y=aX$，则$g_Y(t)=g_X(at)$
      • $g_{X+Y}=g_X \cdot g_Y$

数理统计的基本概念

  • 简单随机抽样
  • 经验分布函数
  • 三大抽样分布
    • $\chi^2$（卡方）分布
      • 定义：$Xi \sim N(0,1)$独立同分布，$\chi ^2 = \sum{i=1}^n X_i^2$服从自由度为n的$\chi^2$分布，记作$\chi^2 \sim \chi^2(n)$
      • 性质：
        • 若$\chi^2 \sim \chi^2(n)$，则$E\chi^2 = n$，$D\chi^2 = 2n$
        • 若$X \sim \chi^2(n_1),Y \sim \chi^2(n_2)$且X, Y相互独立，则$X+Y \sim \chi^2(n_1+n_2)$
    • t分布
      • 定义：$X \sim N(0,1),Y \sim \chi^2(n)$相互独立，$T=\frac {X} {\sqrt{Y/n}}$服从自由度为n的t分布，记作$T \sim t(n)$
    • F分布
      • 定义：$X \sim \chi^2(n_1),Y\sim\chi^2(n_2)$相互独立，$F=\frac {X/n_1} {Y/n_2}=\frac{n_2X}{n_1Y}$服从第一自由度为$n_1$，第二自由度为$n_2$的F分布，记作$F\sim F(n_1,n_2)$
    • $\alpha$分位数：使得$P{X \geq k }=\alpha$的k
  • 统计量
    • 样本均值$\overline{X} = \frac 1 n \sum_{i=1}^n X_i$
    • 样本方差$S^2 = \frac 1 {n-1} \sum_{i=1}^n (X_i - \overline{X})^2$
    • 样本标准差（ 样本均方差）$S = \sqrt{S^2}$
    • 样本k阶原点矩$ak = \frac 1 n \sum{i=1}^n X_i^k,\ k=1,2,\cdots$
    • 样本k阶中心矩$Ak = \frac 1 n \sum{i=1}^n (X_i-\overline{X})^k,\ k=1,2,\cdots$
    • 顺序统计量
    • 样本中位数
    • 样本极差
  • 抽样分布
    • （设$X_1,X_2,\cdots,X_n$为一组来自总体$N(\mu,\sigma^2)$的样本）
      • 样本均值为$\overline{X}$，满足$\overline{X} \sim N(\mu, \frac {\sigma^2} n)$或$\frac {\overline{X}-\mu} {\sigma/\sqrt{n}} \sim N(0,1)$
      • 样本方差为$S^2$，满足$\frac{n-1}{\sigma^2} S^2 = \frac {1} {\sigma^2}\sum_{i=1}^n(X_i - \overline{X})^2 \sim \chi^2(n-1)$
      • 样本均值与样本方差相互独立
      • $\frac {\overline{X}-\mu} {S/\sqrt{n}} \sim t(n-1)$
    • （设$X_1,X_2,\cdots,X_n$和$Y_1,Y_2,\cdots,Y_n$分别是来自总体$N(\mu_1,\sigma^2)$和$N(\mu_2,\sigma^2)$的两组样本，且相互独立）
      • 样本均值差满足$\frac {\overline{X} - \overline{Y}-(\mu_1-\mu_2)} {S_w\sqrt{1/n_1 + 1/n_2}} \sim t(n_1+n_2-2)$，其中$S_w = \sqrt{\frac {(n_1-1)S_1^2 + (n_2-1)S_2^2} {n_1+n_2-2}}$
      • 样本方差比满足$\frac {S_1^2/\sigma_1^2} {S_2^2/\sigma_2^2}\sim F(n_1-1,n_2-1)$
    • （设$X_1,X_2,\cdots,X_n$为一组来自总体$X$的样本，$EX=\mu$，$DX=\sigma^2 < +\infty$）
      • 当样本容量充分大时，近似有$\frac {\sum_{i=1}^n X_i - n\mu} {\sqrt{n}\sigma} \sim N(0,1)$

参数估计

  • 点估计
    • 矩估计法：把未知参数$\theta$表示为总体（原点）矩$\alphak=EX^k\ (k=1,2,\cdots)$的函数，再用样本矩$a_k=\frac 1 n \sum{i=1}^n x_i^k\ (k=1,2,\cdots)$近似替代总体矩得到$\hat{\theta}$作为未知参数$\theta$的估计。
    • 极大似然估计法：设似然函数$L = \prod_{i=1}^n p(x_i;\theta)$，当给定一组样本$x_1,x_2,\cdots,x_n$时，求出使得似然函数取最大值的$\theta=\hat{\theta}$，$\hat{\theta}$就作为未知参数$\theta$的估计。实际计算时通过微积分方法，解方程$\frac {\mathrm{d}} {\mathrm{d}\theta}\ln L = 0$得驻点后确定最大值。
    • 估计量优良性的标准
      • 无偏性：若$E \hat{\theta} = \theta$，则称$\hat{\theta}$为$\theta$的无偏估计量。（若相互独立地重复多次用无偏估计量进行实际估计，所得估计值的算术平均值与未知参数的真值基本上相同）
      • 有效性：若$\hat{\theta}_1, \hat{\theta}_2$都是$\theta$的无偏估计，若$D\hat{\theta}_1 \leq D\hat{\theta}_2$，则称$\hat{\theta}_1$比$\hat{\theta}_2$有效。若对固定样本容量，$D\hat\theta$达到最小值，则称$\hat\theta$为$\theta$的有效估计量。（$\hat\theta_1$比$\hat\theta_2$在$\theta$附近取值的概率大）
      • 一致性：如果$\hat\theta_n =\hat\theta_n(X_1,X_2,\cdots,X_n)$依概率收敛于$\theta$，则称$\hat\theta_n$是$\theta$的一致估计量。（样本容量越大，越能精确估计出未知参数）
  • 区间估计
    • 对未知参数$\theta$，找到两个统计量$\hat\theta_1,\hat\theta_2$使得对给定$\alpha$有$P{\hat\theta_1 < \theta < \hat\theta_2} = 1-\alpha$，则称$(\hat\theta_1 < \theta < \hat \theta_2)$为$\theta$的置信区间，$1-\alpha$为置信度，$\hat\theta_1$和$\hat\theta_2$分别为置信下限和置信上限。
    • $P{\hat\theta_1 < \theta < \hat\theta_2} = 1-\alpha$表示的含义是随机区间$(\hat\theta_1,\hat\theta_2)$包含$\theta$的概率，而不能理解成未知参数落在区间$(\hat\theta_1,\hat\theta_2)$的概率。
    • 单个正态总体参数的区间估计
      • $\sigma^2$已知，$\mu$的置信区间$(\overline x - u{\alpha/2} \frac \sigma {\sqrt{n}},\overline x + u{\alpha/2} \frac \sigma {\sqrt{n}})$
      • $\sigma^2$未知，$\mu$的置信区间$(\overline x - t{\alpha/2}(n-1) \frac S {\sqrt{n}},\overline x + t{\alpha/2}(n-1) \frac S {\sqrt{n}})$
      • $\sigma^2$的置信区间$(\frac {(n-1)S^2} {\chi^2{\alpha/2}(n-1)}, \frac {(n-1)S^2} {\chi^2{1-\alpha/2}(n-1)})$
    • 两个正态总体参数的区间估计
      • 已知$\sigma1^2=\sigma_2^2$，$\mu_1-\mu_2$的置信区间$(\overline x - \overline y - t{\alpha/2}(n1+n_2-2)S_w\sqrt{\frac 1 {n_1} + \frac 1 {n_2}},\overline x - \overline y + t{\alpha/2}(n_1+n_2-2)S_w\sqrt{\frac 1 {n_1} + \frac 1 {n_2}})$，其中$S_w = \sqrt{\frac {(n_1-1)S_1^2 + (n_2-1)S_2^2} {n_1+n_2-2}}$
      • $\sigma1^2/\sigma_2^2$的置信区间$(\frac {S_1^2/S_2^2} {F{\alpha/2}(n1-1,n_2-1)},\frac {S_1^2/S_2^2} {F{1-\alpha/2}(n_1-1,n_2-1)})$
    • 一般总体均值的区间估计
      • $\sigma^2$已知，当n充分大时，$\mu$的置信区间$(\overline x - u{\alpha/2} \frac \sigma {\sqrt{n}},\overline x + u{\alpha/2} \frac \sigma {\sqrt{n}})$，置信度近似为$1-\alpha$
    • (0-1)分布参数的区间估计
      • 当n充分大时，$p$的置信区间$(\frac {-b-\sqrt{b^2-4ac}} {2a},\frac {-b+\sqrt{b^2-4ac}} {2a})$，其中$a=n+u{\alpha/2}^2$，$b=-(2n\overline x + u{\alpha/2}^2)$，$c=n\overline x^2$，置信度近似为$1-\alpha$

假设检验

  • 假设检验方法：假定假设$H_0$正确→构造小概率事件A→一次试验里事件A未出现（出现）→接受（拒绝接受）假设$H_0$

  • 单个正态总体参数的显著性检验

    • （设$x_1,x_2,\cdots,x_n$为取自正态总体$N(\mu,\sigma^2)$的一个容量为n的样本，$\mu_0,\sigma_0$为已知常数）
    • u检验（$\sigma^2$已知，检验$\mu$）
      • 已知$\sigma^2=\sigma_0^2$，检验$H_0:\mu = \mu_0$
        • ①假定$H_0$成立
        • ②选取统计量$u = \frac {\overline X - \mu_0} {\sigma_0/\sqrt{n}} $，并计算统计量$u$的值
        • （设$\widetilde{u} = \frac {\overline X - \mu} {\sigma_0/\sqrt{n}} \sim N(0,1)$，若假设成立，有$u = \widetilde{u}$）
        • ③对给定显著性水平$\alpha$，找到使得$P{|\widetilde u| \geq u{\alpha/2} } = \alpha$的临界值$u{\alpha/2}$【双尾检验】
        • （若假设成立，$P{u \geq u{\alpha}} = P{\widetilde u \geq u{\alpha}} = \alpha$，故$u \geq u_{\alpha}$是小概率事件）
        • ④检查小概率事件$|u| \geq u{\alpha/2} $是否发生，并作判断（即置信区间为$(-u{\alpha/2},u_{\alpha/2}) $）
      • 已知$\sigma^2=\sigma_0^2$，检验$H_0:\mu \leq \mu_0$
        • ①假定$H_0$成立
        • ②选取统计量$u = \frac {\overline X - \mu_0} {\sigma_0/\sqrt{n}}$，并计算统计量$u$的值
        • （设$\widetilde{u} = \frac {\overline X - \mu} {\sigma_0/\sqrt{n}} \sim N(0,1)$，若假设成立，有$u \leq \widetilde{u}$）
        • ③对给定显著性水平$\alpha$，找到使得$P{\widetilde{u} \geq u{\alpha} } = \alpha$的临界值$u{\alpha}$【单尾检验】
        • （若假设成立，$P{u \geq u{\alpha}} \leq P{\widetilde u \geq u{\alpha}} = \alpha$，故$u \geq u_{\alpha}$是小概率事件）
        • ④检查小概率事件$u \geq u{\alpha}$是否发生，并作判断（即置信区间为$(-\infty,u{\alpha})$）
      • 已知$\sigma^2=\sigma_0^2$，检验$H_0:\mu \geq \mu_0$
        • ①假定$H_0$成立
        • ②选取统计量$u = \frac {\overline X - \mu_0} {\sigma_0/\sqrt{n}}$，并计算统计量$u$的值
        • （设$\widetilde{u} = \frac {\overline X - \mu} {\sigma_0/\sqrt{n}} \sim N(0,1)$，若假设成立，有$u \geq \widetilde{u}$）
        • ③对给定显著性水平$\alpha$，找到使得$P{\widetilde{u} \leq -u{\alpha} } = \alpha$的临界值$u{\alpha}$【单尾检验】
        • （若假设成立，$P{u \leq -u{\alpha}} \leq P{\widetilde{u} \leq -u{\alpha} } = \alpha$，故$u \leq -u_{\alpha}$是小概率事件）
        • ④检查小概率事件$u \leq -u{\alpha}$是否发生，并作判断（即置信区间为$(-u{\alpha},+\infty)$）
    • t检验（$\sigma^2$未知，检验$\mu$）
      • 未知$\sigma^2$，检验$H_0:\mu = \mu_0$
        • ①……
        • ②……统计量$t = \frac {\overline X - \mu_0} {S/\sqrt{n}} $……
        • （……$\widetilde{t} = \frac {\overline X - \mu} {S/\sqrt{n}} \sim t(n-1)$……）
        • ③……临界值$t_{\alpha/2}(n-1)$【双尾检验】
        • （……）
        • ④……小概率事件$|t| \geq t{\alpha/2}(n-1) $……（即置信区间为$(-t{\alpha/2}(n-1),t_{\alpha/2}(n-1)) $）
      • 未知$\sigma^2$，检验$H_0:\mu \leq \mu_0$
        • 略（置信区间为$(-\infty,t_{\alpha/2}(n-1))$）
      • 未知$\sigma^2$，检验$H_0:\mu \geq \mu_0$
        • 略（置信区间为$(-t_{\alpha/2}(n-1),+\infty)$）
    • $\chi^2$检验（$\mu$未知，检验$\sigma^2$）
      • 未知$\mu$，检验$H_0:\sigma^2=\sigma_0^2$
        • ①……
        • ②……统计量$\chi^2 = \frac {(n-1)S^2} {\sigma_0^2} $，……
        • （……$\widetilde\chi^2 = \frac {(n-1)S^2} {\sigma^2} \sim \chi^2(n-1)$……）
        • ③……临界值$\chi{1-\alpha/2}^2 (n-1)$和$\chi{1-\alpha/2}^2 (n-1)$【双尾检验】
        • （……）
        • ④……小概率事件$\chi^2 \leq \chi{1-\alpha/2}^2 (n-1)$或$\chi^2 \geq \chi{\alpha/2}^2 (n-1)$……（即置信区间为$(\chi{1-\alpha/2}^2 (n-1),\chi{\alpha/2}^2 (n-1)) $）
      • 未知$\mu$，检验$H_0:\sigma^2 \leq \sigma_0^2$
        • 略（置信区间为$(-\infty,\chi_{1-\alpha/2}^2 (n-1))$）
      • 未知$\mu$，检验$H_0:\sigma^2 \geq \sigma_0^2$
        • 略（置信区间为$(-\chi_{\alpha/2}^2 (n-1),+\infty)$）

  • 两个正态总体参数的显著性检验

    • （设$X1,X_2,\cdots,X{n1}$为取自正态总体$N(\mu_1,\sigma_1^2)$的一个容量为$n_1$的样本，$Y_1,Y_2,\cdots,Y{n_2}$为取自正态总体$N(\mu_2,\sigma_2^2)$的一个容量为$n_2$的样本）

    • t检验（续）

      • 未知$\sigma_1^2,\sigma_2^2$，但已知$\sigma_1^2=\sigma_2^2$，检验$H_0:\mu_1 = \mu_2$
        • ①……
        • ②……统计量$t = \frac {\overline X - \overline Y} {S_w\sqrt{1 /{n_1} + 1/{n_2}}} $，……
        • （……$\widetilde t = \frac {\overline X - \overline Y - \mu_1-\mu_2} {S_w\sqrt{1 /{n_1} + 1/{n_2}}} \sim t(n_1+n_2-2)$……）
        • ③……临界值$t_{\alpha/2}(n_1+n_2-2)$【双尾检验】
        • （……）
        • ④……小概率事件$|t| \geq t{\alpha/2}(n_1+n_2-2) $……（即置信区间为$(-t{\alpha/2}(n1+n_2-2),t{\alpha/2}(n_1+n_2-2)) $）
      • 未知$\sigma_1^2,\sigma_2^2$，但已知$\sigma_1^2=\sigma_2^2$，检验$H_0:\mu_1 \leq \mu_2$
        • 略（置信区间为$(-\infty,t_{\alpha}(n_1+n_2-2)) $）

    • F检验（续）

      • 未知$\mu_1,\mu_2$，检验$H_0:\sigma_1^2=\sigma_2^2$

        • ①……
        • ②……统计量$F=\frac {S_1^2}{S_2^2}$，……
        • （……$\widetilde F=\frac {S_1^2/\sigma_1^2} {S_2^2/\sigma_2^2}\sim F(n_1-1,n_2-1)$……）
        • ③……临界值$F{1-\alpha/2} (n_1-1,n_2-1)$和$F{\alpha/2} (n_1-1,n_2-1)$【双尾检验】
        • （……）
        • ④……小概率事件$F \leq F{1-\alpha/2} (n_1-1,n_2-1)$或$F \geq F{\alpha/2} (n1-1,n_2-1)$……（即置信区间为$(F{1-\alpha/2} (n1-1,n_2-1),F{\alpha/2} (n_1-1,n_2-1)) $）

      • 未知$\mu_1,\mu_2$，检验$H_0:\sigma_1^2\leq\sigma_2^2$

        • 略（置信区间为$(-\infty,F_{\alpha} (n_1-1,n_2-1)) $）
